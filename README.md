# MrDoc

## 介绍
一个简单的文档写作系统

## 软件架构
基于Python Web框架Django构建。

语言：Python 3

后端：Django 2.1

前端UI：LayUI 2.5.4

文档编辑器：Editormd

社交分享：Share.js

科学公式：Katex

## 安装教程

1. 下载并解压；
2. 如果需要使用其他数据库，请修改；
3. 运行python manage.py makemigrations&&python manage.py migrate 创建和执行数据库迁移；
4. 运行python manage.py createsuperuser 创建管理员账号；
